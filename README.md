# GUESSR 💕

A fast-paced, real-time 1v1 number guessing duel game with chaotic modifiers, power-ups, and a romantic neon aesthetic. No installs, no accounts needed.

## ✨ Features

- **1v1 Multiplayer**: Fast and seamless room creation via 4-letter codes.
- **Modifiers System**: Draft visual and gameplay modifiers every round. Play in *Neon*, *Blind Mode*, use *Telepathy*, *Peek* hints, or try the *Heartbeat* modifier.
- **Series & Streaks**: Best-of-X modes (1, 3, 5, 7) with round streak tracking.
- **Comeback Mechanic**: Automatically shrinks the guessing range by 50% if you fall behind.
- **Admin Debug Panel**: Password-protected side panel (Code `3109`) for local testing, adding bots (`DebugBot`), and live score editing.
- **Mobile First**: PWA-ready layout that feels native on phones and tablets.

## 🚀 Getting Started

The project is static and serverless. You can deploy it to GitHub Pages or any static file host out of the box.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sistineinfinitia-prog/guessy.git
   ```
2. Open `index.html` in your browser.
3. Use the **Secret Access Code**: Enter `3109` in the Join Room code box to open the admin/debug panel.
